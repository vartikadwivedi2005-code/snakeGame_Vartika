🐍 Snake Game
A Minimalist Arcade Experience

📖 About the Project
Snake Game is a web-based project developed using HTML5, CSS3, and JavaScript. The goal was to create a seamless user experience where logic meets design.

In this game, the player controls a growing line (the "Snake") that becomes a primary obstacle to itself. It is a test of reflexes, spatial awareness, and strategy.

Why I Built This:
Logic Mastery: To practice handling arrays and coordinate systems in JavaScript.

Performance: Using the HTML5 Canvas API to ensure high frame rates and zero lag.

Web Design: To create a visually pleasing, "dark-mode" aesthetic for a classic game.

🎮 Gameplay Mechanics
Growth System: Every time the Snee consumes food, it increases in length.

Collision Detection: The game ends if the Snee hits the boundary of the canvas or intersects with its own body.

Score Tracking: A real-time counter keeps track of your progress.
